## Bedroom Spring Cleaning Checklist

- [ ] Dismantle dressers and bed headboard.
- [ ] Remove curtains and blinds.
- [ ] Clean windows.
- [ ] Organize closet.
- [ ] Wash pillows.
- [ ] Wash bed linens.
- [ ] Flip mattresses.
- [ ] Vacuum floors.
- [ ] Disassemble lamps.
- [ ] Dust cabinets.
- [ ] Disassemble shelves and racks.
- [ ] Clean doors and frames.
- [ ] Wipe walls and corners.
- [ ] Wipe baseboards.
